To import this custom module use the import command 'from cachy import cache'.
Usage of module:
- cache.initialize() - runs the modules setup
- cache.deposit(data=<data to save>, id=<identifier>) - caches data under the identifier name
- <variable> = cache.withdraw.(id=<identifier>) - fetches cached data under the identifier name
- cache.save(data=<data to save>, id=<identifier>) - save the data and caches under the identifier name
- cache.save_item(id=<identifier>) - saves an exisiting cached data via identifier name
- <variable> = cache.load(id=<identifier>) - loads saved data into cache then fetches the same data from cache under the identifer name
- cache.destroy(id=<identifier>) - destroys cache item under the identifier name
- cache.destroy_item(id=<identifier>) - destroys the cached item and the saved item under the indentifier name
- cache.unsave(id=<identifier>) - destroys the saved item under the identifier name
- cache.clear() - clears the cache
- cache.clear_all() - clear the cache and the saved file cache

Always run cache.initialize before module usage to prevent errors and issues.
